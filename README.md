## Installation

### Unix

Dependencies:

- GNU Stow
- cURL

```bash
chmod +x setup.sh
./setup.sh [-n: dry run]
```

### NT

Dependencies:

- Administrator PowerShell window

```powershell
Set-ExecutionPolicy Unrestricted
./PowerShell/Scripts/firstRun/firstRun.ps1
```
